# 中医基础理论核心学说深度讲解

> 写给零基础学生的中医理论入门。不死记硬背，只讲明白道理。

---

## 开篇：中医理论的"操作系统"

学中医，最怕一上来就背。"肝主疏泄""脾主运化""木火土金水"……背了一堆，不知道为什么，考完就忘。

换个思路——**先搞懂逻辑，再记细节**。

中医理论就像一台电脑的操作系统，由几个核心模块组成：

| 模块 | 回答的问题 | 类比 |
|------|-----------|------|
| 阴阳学说 | 世界的基本规律是什么？ | 操作系统的底层代码 |
| 五行学说 | 事物之间怎么相互影响？ | 程序之间的通信协议 |
| 藏象学说 | 人体有哪些"部门"，各管什么？ | 硬件配置清单 |
| 气血津液 | 人体靠什么运转？ | 电力、网络、冷却系统 |
| 经络学说 | 这些东西走哪条路？ | 电路板上的线路 |
| 病因病机 | 为什么会出故障？怎么发展的？ | 故障诊断手册 |

我们的学习路线：**先哲学框架（阴阳→五行），再人体地图（脏腑→气血→经络），最后实战（病因病机→辨证治疗）。**

每一章都建立在前一章的基础上，环环相扣。搞懂了前面的，后面的自然就通了。

# 第一章：阴阳学说

> **本章你将学到：** 阴阳到底是什么、阴阳之间的四种关系、中医怎么用阴阳看病治病。

## 一、阴阳是什么？

### 从你已经知道的开始

你每天都在跟阴阳打交道，只是没人告诉你那叫阴阳：

- 白天亮，晚上暗
- 夏天热，冬天冷
- 火往上烧，水往下流
- 运动时兴奋发热，睡觉时安静凉爽

古人观察了这些现象，发现一个规律：**世界上的事情，总是成对出现的，而且这一对之间存在"你多我少、你少我多"的关系。**

他们给这种关系取了个名字——阴阳。

- **阳：** 明亮的、温热的、活动的、向上的、兴奋的那一面
- **阴：** 黑暗的、寒凉的、安静的、向下的、抑制的那一面

### 最关键的一点

**阴阳不是两种"东西"，而是一种看问题的方法。**

举个例子：水是阴还是阳？

你可能会说"水是阴"。相对于火来说，没错。但是——

- 热水和冷水比呢？热水是阳，冷水是阴。
- 流动的水和静止的水比呢？流动的是阳，静止的是阴。

同一个"水"，到底是阴还是阳？**取决于你拿它跟谁比，以及你从哪个角度看。**

这就像问一个人"你是高还是矮"——跟姚明比他矮，跟小孩比他高。高矮不是标签，而是**比较关系**。阴阳也是如此。

### 阴阳可以无限细分

白天属阳，黑夜属阴。但白天内部还能再分：

- 上午：太阳在升，温度在涨 → **阳中之阳**
- 下午：太阳在落，温度在降 → **阳中之阴**

黑夜也能再分：

- 前半夜：越来越深、越来越冷 → **阴中之阴**
- 后半夜：开始向天亮过渡 → **阴中之阳**

可以一直分下去，任何层级里面都还有阴阳。

**关键类比：温度计。** 不是只有"冷"和"热"两个刻度，而是一个连续的光谱，从极寒到极热，中间无数过渡。阴和阳是光谱的两端，万事万物落在中间某个位置。

## 二、阴阳的四大关系

阴和阳之间怎么"相处"？四种关系：

### 1. 对立制约——"互相管着对方"

阴和阳是对立的，一方会"压制"另一方，不让它太过分。

**恒温器类比：** 你家空调设定 25 度——室温高了就制冷，低了就制热。制冷和制热互相制约，房间才能保持舒适温度。如果只有制热没有制冷？温度一路飙升。

人体也一样。产热的力量（阳）和降温的力量（阴）互相压着，体温才稳定在 37 度左右。

> **对立制约 = 恒温器。两股力量互相牵制，系统才能稳定。**

### 2. 互根互用——"谁也离不开谁"

阴离开阳就不存在，阳离开阴也不存在。

想一下：如果世界上永远没有黑夜，那"白天"这个词还有意义吗？没有。"白天"是相对于"黑夜"才成立的。如果从来没有安静过，你就不会觉得"吵"。

**有阴才知道什么是阳，有阳才知道什么是阴。它们互相定义对方。**

更深一层——它们还**互相支撑**。

**蜡烛类比：** 蜡是阴（物质基础），火焰是阳（功能、能量）。没有蜡，火烧什么？火灭了。没有火，蜡有什么用？就是一块冷蜡。**阴是阳的"原料"，阳是阴的"动力"。**

> **互根互用 = 蜡烛。蜡供火烧，火让蜡有价值。**

### 3. 消长平衡——"此消彼长，动态平衡"

阴阳不是静止的，一方增长时另一方在减少，但正常情况下维持在平衡范围内。

**四季类比：** 冬至到夏至，白天越来越长、气温升高——阳长阴消。夏至到冬至，白天变短、气温降低——阴长阳消。但不会热到烤化地球，也不会冷到绝对零度，在一个范围里来回摆动。

就像钟摆——左摆右摆，始终围着中间那个点晃。

- **正常消长 = 健康。** 白天精力充沛（阳长），晚上安静入睡（阴长），第二天满血复活。
- **消长失控 = 生病。** 阳一直长、阴一直消，刹不住车——出问题了。

> **消长平衡 = 钟摆。在范围内摆动是正常，摆过头就是失调。**

### 4. 相互转化——"走到极端就翻转"

阴到极点会变成阳，阳到极点会变成阴。消长是**量变**，转化是**质变**。

**烧水类比：** 给水加热——40 度、60 度、80 度……还是水，只是越来越热（消长）。100 度！水变成蒸汽——液体（阴）变成气体（阳）。这就是**转化**。

中文四个字概括：**物极必反。**

再举个例子：一个人拼命加班不睡觉（阳亢到极点），最后突然倒下昏睡不醒（转化为阴）。不是休息，是身体强制关机——阳极转阴。

转化的前提：到了极点 + 阴中有阳的种子、阳中有阴的种子。太极图里黑色中的白点、白色中的黑点就是这个种子。

> **相互转化 = 水烧开变蒸汽。量变积累到极点，质变发生。**

### 四大关系总览

| 关系 | 一句话概括 | 生活类比 |
|------|-----------|---------|
| 对立制约 | 互相管着 | 空调恒温器 |
| 互根互用 | 谁也离不开谁 | 蜡烛的蜡和火 |
| 消长平衡 | 此消彼长，动态平衡 | 四季交替 / 钟摆 |
| 相互转化 | 走到极端就翻转 | 水烧开变蒸汽 |

四条不是孤立的：**对立的双方互相依存，在制约中此消彼长，积累到极点时发生质变翻转。**

## 三、阴阳在中医里怎么用？

### 1. 用阴阳给人体"画地图"

| 阳 | 阴 | 理解方式 |
|----|----|---------|
| 上半身 | 下半身 | 热气上升（阳在上），重物下沉（阴在下） |
| 体表 | 体内 | 皮肤接触外界、活动多（阳）；内脏藏在里面、安静（阴） |
| 背部 | 腹部 | 背对着太阳晒（阳）；腹部朝内保护（阴） |
| 六腑 | 五脏 | 腑传化不止（阳）；脏藏精不泻（阴） |

### 2. 用阴阳看生理：气和血的故事

- **气 = 阳的代表。** 推动、温煦、保护的力量，看不见摸不着但在干活。
- **血 = 阴的代表。** 营养、滋润的物质，看得见、有实体。

**气推血行（阳推动阴）：** 气是血的动力。没有气推，血走不动会瘀住。像水管没有水泵，水流不起来。

**血载气运（阴承载阳）：** 血是气的载体。气必须附在血上才能到达全身。没有血，气就散了。

中医原话：**"气为血之帅，血为气之母。"**

### 3. 用阴阳辨病：怕冷还是怕热？

**怕冷 → 寒证：**
- **阴盛：** 阴太多，把阳压住了。冬天来了，太阳被压制。
- **阳虚：** 阳太少，管不住阴。暖气坏了，屋里自然冷。

**怕热 → 热证：**
- **阳盛：** 阳太多。大夏天，你当然热。
- **阴虚：** 阴太少，管不住阳。水壶的水快烧干了，翻滚得特别厉害——虚热。

### 4. 用阴阳治病：治疗就是"纠偏"

> **热者寒之，寒者热之。实则泻之，虚则补之。**

- 阳盛发热 → 寒凉药**清热**（削减多出来的）
- 阴盛怕冷 → 温热药**散寒**（散掉多出来的）
- 阳虚怕冷 → **补阳**（补上不够的）
- 阴虚发热 → **滋阴**（补上不够的）

多了就削减，少了就补充。

### 5. 完整病例

**场景：** 30 岁女性，手脚心发热，晚上睡觉出汗，嘴巴干，心烦睡不好。

**分析过程：**

1. **辨寒热：** 手脚心热、心烦 → 热证。
2. **判虚实：** 不是全身大热，而是手脚**心**热（五心烦热）；夜间盗汗；口干但不狂灌水 → 虚热，不是实热。
3. **定病机：** 阴液不足，管不住阳气，阳气浮越外散。水壶里的水快干了，那一点水翻滚得厉害。
   - 浮到手脚心 → 手脚心热
   - 浮到体表逼汗外泄 → 盗汗
   - 阴液亏损 → 口干
   - 阳气上扰心神 → 心烦失眠
4. **治法：** 滋阴。把水加回去，水多了自然镇住火。经典方：**六味地黄丸**。不需要猛烈清热——火不是真的多，是水太少的假象。

> **观察（症状）→ 判断（阴虚内热）→ 治疗（滋阴）**，一条完整的链条。

### 本章小结

阴阳是中医的**底层操作系统**。它是一副"动态平衡"的眼镜——戴上它看世界，万事万物都是对立双方在制约中共存、在消长中平衡、在极端时转化。中医的全部工作，就是发现这个平衡在哪里偏了，然后纠回来。

但阴阳只能把事物分成"两面"。如果我想描述更复杂的关系——比如五个脏腑之间怎么相互影响——光靠二分法就不够用了。这时候就需要下一个工具：**五行。**

# 第二章：五行学说

> **本章你将学到：** 五行到底是什么、相生相克怎么运作、五行怎么对应五脏、临床怎么用。

## 一、五行是什么？

### 最常见的误解

很多人一听"金木水火土"，就觉得这是古人在说世界由五种物质组成。

**不是的。**

五行的"行"字，意思是"运行、运动"。五行不是五种"元素"，而是五种**运动状态**，或者说五种**功能模式**。

古人观察自然界，发现万事万物虽然千变万化，但可以归纳成五种典型的"做事风格"：

| 五行 | 核心特性 | 一句话理解 | 自然现象 |
|------|---------|-----------|---------|
| **木** | 生长、舒展、向上 | 像春天的树，拼命往上长 | 春天，万物萌发 |
| **火** | 炎热、上升、光明 | 像夏天的太阳，热烈奔放 | 夏天，热力鼎盛 |
| **土** | 承载、化生、居中 | 像大地，接受一切、孕育一切 | 长夏（夏末），万物成熟 |
| **金** | 收敛、肃降、清洁 | 像秋天的风，把多余的扫干净 | 秋天，收获肃杀 |
| **水** | 润下、闭藏、寒冷 | 像冬天的冰，把一切封存起来 | 冬天，万物蛰伏 |

### 用一天来理解五行

如果一年四季太抽象，想想一天：

- **早晨（木）：** 起床、伸展、开始活动——生发向上
- **中午（火）：** 精力最旺、最活跃——热烈巅峰
- **下午（土）：** 吃完午饭消化中，身体在转化营养——承载运化
- **傍晚（金）：** 开始收工、减少活动——收敛下降
- **深夜（水）：** 进入深度睡眠、身体修复——闭藏休息

一天就是一个完整的五行循环。每个阶段自然过渡到下一个，停不下来。

## 二、五行的两大关系

五行之间不是各管各的，它们有两套"管理制度"。

### 1. 相生——"母子关系"，互相促进

**木 → 火 → 土 → 金 → 水 → 木**……循环往复。

"生"就是促进、滋养。生我的是我的"母亲"，我生的是我的"孩子"。

怎么理解？

- **木生火：** 木头能燃烧生出火。（春天的生长积蓄能量，到夏天爆发）
- **火生土：** 火燃烧后变成灰烬，灰烬归于大地。（热量促进万物成熟，果实落地归土）
- **土生金：** 矿石（金属）从土里挖出来。（大地孕育收获）
- **金生水：** 金属表面会凝结水珠（想想冰镇可乐罐外的水滴）。（秋天的收敛为冬天的闭藏做准备）
- **水生木：** 树木需要水才能生长。（冬天积蓄的能量供春天萌发）

**相生的意义：** 每一行都为下一行提供条件，形成一个良性循环。任何一环断了，后面的都受影响。

### 2. 相克——"制约关系"，防止一方独大

**木 → 土 → 水 → 火 → 金 → 木**……循环制约。

"克"就是制约、抑制。

- **木克土：** 树根扎入土壤，破土而出。（木的舒展打破土的固守）
- **土克水：** 兵来将挡，水来土掩。（土能筑堤挡水）
- **水克火：** 水能灭火。
- **火克金：** 火能熔化金属。
- **金克木：** 斧头（金属）能砍树。

**相克的意义：** 如果只有相生没有相克，某一行一路猛长，没人管它，系统就失衡了。相克就像公司里的审计部门——不是来搞破坏的，而是防止某个部门膨胀失控。

### 相生相克联合起来看

每一行都同时处于四种关系中：

- **生我者**（母）—— 滋养我
- **我生者**（子）—— 我滋养它
- **克我者** —— 制约我
- **我克者** —— 我制约它

比如"火"：

| 关系 | 对象 | 含义 |
|------|------|------|
| 生我者（母） | 木 | 木生火，木是火的能量来源 |
| 我生者（子） | 土 | 火生土，火为土提供条件 |
| 克我者 | 水 | 水克火，水对火有制约 |
| 我克者 | 金 | 火克金，火对金有制约 |

**每一行都被"养着"也被"管着"，既能影响别人，也被别人影响。这就是一个自我调节的系统。**

## 三、五行配五脏——为什么这样配？

| 五行 | 五脏 | 配对逻辑 |
|------|------|---------|
| 木 | **肝** | 肝主疏泄，喜欢舒展条达，就像春天的树木生长——憋屈了就"肝气郁结" |
| 火 | **心** | 心主血脉，是人体的"发动机"，温暖全身，热力最强——像夏天的太阳 |
| 土 | **脾** | 脾主运化，把食物变成营养输送全身，是"后天之本"——像大地承载万物、化生万物 |
| 金 | **肺** | 肺主肃降，呼吸清气、排出浊气，喜欢清洁——像秋风扫落叶 |
| 水 | **肾** | 肾主藏精，是生命的"储备库"，主管水液代谢——像冬天封藏一切 |

不是随便配的，是因为这个脏的**功能特点**和这一行的**运动风格**高度吻合。

一旦配上了，五行之间的相生相克关系就自动变成了**脏腑之间的关系**：

- **肝木生心火：** 肝的疏泄功能正常，气血畅通，心才能好好推动血液。
- **心火生脾土：** 心的阳气温暖脾胃，脾才能正常消化。
- **脾土生肺金：** 脾把营养运上去（水谷精微上输于肺），肺才有原料造气。
- **肺金生肾水：** 肺主肃降，把水液和清气降到肾，供肾封藏。
- **肾水生肝木：** 肾精滋养肝血，肝才有"原料"去疏泄。

## 四、五行在中医里怎么用？

### 解释"连锁反应"

生活中有个很常见的现象：**一生气就吃不下饭。**

用五行解释：

1. 生气 → 肝气郁结（肝木出了问题）
2. 肝属木，脾属土，木克土
3. 肝气太盛 → **过度克制脾土** → 脾的运化功能被压制
4. 结果：食欲下降、胃胀、不想吃东西

这就是中医说的**"肝木乘脾土"**（"乘"就是过度克制）。

如果你只盯着"吃不下饭"去治脾胃，效果不好。因为**根源在肝**，得先疏肝。

### 治疗思路

五行给了中医两条重要的治疗原则：

**虚则补其母：** 如果某一行虚弱了，去补养它的"母亲"。

- 肺（金）虚 → 补脾（土），因为土生金
- 中医实践：肺气虚的人反复感冒，经常不是直接补肺，而是先健脾——脾把营养运上去了，肺自然就强了

**实则泻其子：** 如果某一行太旺了，去泻它的"孩子"。

- 肝（木）火旺 → 不一定直接泻肝，可以泻心（火），因为木生火——让火这个出口畅通了，木的压力自然减轻

### 病例：肝火犯胃

**场景：** 一个人最近工作压力大，情绪暴躁。开始出现：口苦、反酸、胃痛、烧心，吃辛辣的更严重。

**分析：**

1. 情绪暴躁 → 肝火旺（肝木太盛）
2. 胃属六腑配脾土 → 木克土 → 肝火横逆犯胃
3. 口苦是肝胆火热上蒸，反酸烧心是胃气被肝火逼得上逆

**治法：** 疏肝泻火 + 和胃降逆。不能光治胃，必须同时管肝。

经典方：**左金丸**（黄连泻火、吴茱萸引火下行兼暖胃）。

### 本章小结

五行是阴阳的"升级版"——阴阳把世界分成两面，五行把世界分成五种运动模式。通过相生相克，五行构建了一个**自我调节的网络**。配上五脏后，这个网络就变成了理解脏腑关系的核心工具。

记住：**五行不是迷信，而是一套描述"系统内多方互动"的语言。** 现代系统论讲的"正反馈""负反馈""动态平衡"，古人用相生相克就说清楚了。

接下来，我们进入人体——看看五脏六腑到底各自干什么活。

# 第三章：藏象学说（脏腑）

> **本章你将学到：** "藏象"是什么意思、五脏各自的核心功能、六腑的角色、脏腑之间的关系。

## 一、"藏象"是什么？

"藏"是藏在体内的脏腑，"象"是表现在外面的征象。

翻译一下：**通过观察外在的表现，推断内在脏腑的状态。**

比如：你看到一个人面色发黄、没食欲 → 推断他的**脾**可能有问题。你不需要切开肚子看脾脏，外在的"象"已经告诉你了。

### 最重要的提醒

**中医的"脏"≠ 解剖学上的器官。**

中医说的"心"，不只是那个拳头大小的肌肉泵。它是一个**功能系统**，包含了"推动血液循环"和"主管精神意识"两大功能。

中医说的"肾"，也不只是那两个过滤血液的器官。它是"生长发育、生殖、水液代谢、骨骼发育"等一整套功能的总称。

**把中医的脏理解成"部门"而非"器官"，就对了。** 一个部门可能涉及多个物理位置，但它负责一类工作。

---

## 二、五脏逐一讲解

### 心——CEO 兼中央处理器

**核心功能：**

| 功能 | 含义 | 大白话 |
|------|------|--------|
| 主血脉 | 推动血液在脉管中运行 | 心是血液循环的"水泵" |
| 主神明 | 主管精神、意识、思维活动 | 心是大脑的"指挥中心" |

**为什么心"主神明"？**

你可能会疑惑：精神活动不是大脑管的吗？在中医体系里，"心"承担了部分现代医学归给大脑的功能。你不需要纠结这个矛盾——只需要记住，在中医话语体系里，说一个人"心神不宁"，是指他精神状态不好，不是说他心脏有病。

**生活类比：** 心就像公司的 CEO——既要管"业务运转"（血液循环），又要管"战略决策"（精神意识）。CEO 状态好，公司运转有序、决策清晰；CEO 崩了，整个公司一团乱。

**常见失调表现：**
- 心血不足：面色苍白、心悸、健忘、失眠多梦
- 心火亢盛：口舌生疮、心烦、失眠、小便赤热

**心开窍于舌，其华在面。** 看一个人舌尖红不红、面色好不好，就能初步判断心的状态。

---

### 肺——空气净化系统 + 分发中心

**核心功能：**

| 功能 | 含义 | 大白话 |
|------|------|--------|
| 主气、司呼吸 | 管理全身的气，负责呼吸 | 肺是"气的总管" |
| 主宣发肃降 | 向上向外散布 + 向下清肃 | 既往外"喷"，又往下"降" |
| 主皮毛 | 管理皮肤、毛发、汗孔 | 身体的"第一道防线" |
| 通调水道 | 参与水液代谢 | 水液运输的"上游调度" |

**宣发和肃降怎么理解？**

想象一台中央空调：
- **宣发 = 吹出去。** 把卫气和津液向上、向外输送到体表和全身——像空调把暖风吹到每个房间。
- **肃降 = 吸进来、沉下去。** 把吸入的清气向下输送，把体内的废水废气向下推——像空调的回风系统把废气排掉。

两个方向一上一下，配合着来，气的运行和水的代谢才正常。

**常见失调表现：**
- 肺气虚：声低气短、容易感冒、自汗
- 肺失宣降：咳嗽、气喘、鼻塞、痰多

**肺开窍于鼻。** 所以感冒鼻塞、流鼻涕，中医首先想到肺。

---

### 脾——消化工厂 + 物流中心

**核心功能：**

| 功能 | 含义 | 大白话 |
|------|------|--------|
| 主运化 | 把食物变成营养，输送到全身 | 消化吸收 + 物流配送 |
| 主统血 | 统摄血液在脉管内运行，不外溢 | 防止出血 |
| 主肌肉、四肢 | 肌肉的营养来源 | 肌肉有没有力气，跟脾有关 |
| 主升清 | 把营养"升"到上面去（心、肺、头） | 对抗地心引力的力量 |

**脾为什么这么重要？**

你吃的每一口饭，都要靠脾来"加工"和"发货"。如果脾罢工了：
- 食物消化不了 → 腹胀、没食欲
- 营养发不出去 → 全身没力气、肌肉松软
- 水湿运不走 → 身体沉重、痰多、腹泻
- 统血失职 → 皮下出血、月经量多

所以中医说脾是**"后天之本"**——你出生后活下去的本钱全靠脾。（"先天之本"是肾，后面讲。）

**生活类比：** 脾就像一个同时运营着食品加工厂和快递公司的企业。工厂负责把原材料（食物）加工成成品（营养），快递公司负责把成品送到全身各地。哪个环节出问题，下游都会断供。

**脾开窍于口，其华在唇。** 嘴唇淡白没血色、吃东西没味道——看看脾。

---

### 肝——交通指挥中心 + 血库

**核心功能：**

| 功能 | 含义 | 大白话 |
|------|------|--------|
| 主疏泄 | 保持气机通畅、调节情绪 | 全身气的"交通指挥" |
| 主藏血 | 储存血液、调节血量 | 身体的"血库" |

**"疏泄"为什么这么重要？**

"疏泄"就是"疏通、发泄"。肝负责让全身的气运行得通畅。

气通畅了：
- 情绪舒畅（不憋屈）
- 消化正常（气推动脾胃工作）
- 胆汁分泌正常
- 月经通畅

气不通畅了（肝气郁结）：
- 心情烦躁或抑郁（"气"得不行）
- 两肋胀痛（气堵在那里）
- 消化不好（肝木克脾土）
- 月经不调（气滞则血瘀）

**你身边一定有这种人：** 性格特别要强，什么事都往心里憋，动不动就叹气、两肋不舒服——典型的肝气郁结。

**藏血怎么理解？**

肝就像一个蓄水池。你活动的时候，肝把血释放出去供四肢使用；你休息的时候，血回流到肝里储存。

这就是为什么中医说**"人卧则血归于肝"**——你躺下来休息的时候，血回到肝里"充电"。经常熬夜的人肝血不足，就是因为血一直在外面跑，没时间回来充。

**肝开窍于目。** 眼睛干涩、视力模糊——肝血不足。

---

### 肾——生命银行 + 水利局

**核心功能：**

| 功能 | 含义 | 大白话 |
|------|------|--------|
| 主藏精 | 储存先天和后天之精 | 生命的"储蓄账户" |
| 主水 | 管理全身水液代谢 | 身体的"水利局" |
| 主纳气 | 帮助肺把气"吸到底" | 呼吸的"锚" |
| 主骨、生髓、通脑 | 骨骼发育、大脑充养 | 骨骼和大脑的营养来源 |

**肾精是什么？**

"精"你可以理解为"生命的本钱"。它分两部分：

- **先天之精：** 父母给的，出生时就有，决定了你的体质底子。像银行账户里的**初始存款**。
- **后天之精：** 脾胃从食物中提取的营养，不断补充进来。像你每个月往账户里**存的工资**。

先天之精是有限的，用一点少一点（衰老就是精在减少的过程）。后天之精可以补充，但补充速度比不上挥霍速度的话，还是会亏。

这就是为什么中医说肾是**"先天之本"**——它是生命的起点和根基。

**"主水"怎么理解？**

你喝进去的水，在体内经过一系列的"蒸馏、分配、排泄"过程。肾是这个过程的**总闸门**——有用的水留下来（重吸收），没用的水排出去（变成尿）。

肾阳不足 → 闸门关不住 → 尿频、夜尿多、水肿
肾阴不足 → 闸门关太紧 → 尿少、口干

**肾开窍于耳，其华在发。** 老年人耳聋、头发白——肾精亏了。

## 三、六腑简述

五脏主"藏"（储存精气），六腑主"传化"（接收、消化、传导、排泄）。

如果说五脏是**仓库**，六腑就是**流水线**——东西不能停在里面，必须一直往前走。

| 六腑 | 功能 | 配对的脏（表里关系） |
|------|------|-------------------|
| **胆** | 储存和排泄胆汁，参与消化；主决断 | 肝 |
| **胃** | 接收食物，初步消化（"腐熟水谷"） | 脾 |
| **小肠** | 分清泌浊——把营养和废物分开 | 心 |
| **大肠** | 传导糟粕，排出大便 | 肺 |
| **膀胱** | 储存和排泄尿液 | 肾 |
| **三焦** | 水液和气运行的通道，"决渎之官" | 心包 |

**表里关系** 就是每个脏都有一个腑跟它"配对"，一阴一阳、一里一表。生病时经常互相牵连——比如肝有问题可能影响胆，脾有问题可能影响胃。

## 四、脏腑之间的关系网络

五脏不是各干各的，它们之间有紧密的协作关系。挑几个最重要的讲：

### 心与肾——水火既济

心在上属火，肾在下属水。正常情况下：
- **心火下降** 温暖肾水，让肾不至于太寒
- **肾水上升** 滋养心火，让心不至于太亢

就像一栋楼：楼上的暖气往下送热量，楼下的水管往上送冷水，上下交通，温度适宜。这叫**"水火既济"**。

如果心火降不下去、肾水升不上来：
- 心火亢于上 → 失眠、心烦、口舌生疮
- 肾水寒于下 → 腰膝冷痛、夜尿频

上热下寒同时出现——这就是"心肾不交"。临床非常常见。

### 肝与脾——木与土的恩怨

前面五行那章讲过：木克土 = 肝克脾。

正常情况下，肝的疏泄功能**帮助**脾的运化（适度的制约是促进）。但如果：
- 肝气太旺（生气、压力大）→ **过度克制脾** → 吃不下饭、腹胀腹泻
- 脾太虚弱 → **反被肝欺负**（本来正常的克制，对虚弱的脾来说就受不了了）

临床上一大半的消化问题都跟情绪有关——**这就是肝脾关系在作怪。**

### 肺与肾——金水相生

肺属金，肾属水，金生水。

- **肺的肃降** 把水液和清气往下送到肾
- **肾的纳气** 帮助肺把吸入的气"拉到底"

如果肾虚不能纳气 → 气浮在上面下不来 → 吸气困难、动则气喘。

老年人慢性哮喘、一动就喘——往往不只是肺的问题，根在**肾虚不纳气**。这时候光治肺不够，必须补肾。

### 肝与肾——同源互养

"肝肾同源"——肝藏血，肾藏精。精和血互相转化、互相滋养。

- 肾精充足 → 化生肝血 → 肝的功能正常
- 肝血充足 → 反哺肾精 → 肾的储备不亏

临床常见：更年期女性肝肾阴虚——肾精亏了、肝血也少了，出现头晕、耳鸣、腰酸、眼干、烦躁，补肝补肾往往一起来。

### 本章小结

藏象学说是把人体当成一个由"部门"组成的组织来管理。五脏是核心部门（各有职能），六腑是配套的流水线，它们通过表里关系配对、通过五行关系互动。**理解了每个脏的功能特点和脏腑之间的关系网络，你就拿到了中医的"人体地图"。**

有了地图，还需要知道地图上跑的是什么——接下来讲**气血津液**，人体运转的"燃料和物资"。

# 第四章：气血津液

> **本章你将学到：** 气是什么、血是什么、津液是什么，它们之间怎么配合，出问题会怎样。

## 一、气——看不见的"动力"

### 气到底是什么？

很多人觉得"气"玄乎。其实不玄。

你深呼吸一口——那股从鼻子进入肺的空气，是气的一部分。
你吃完饭感觉有力气了——食物转化出的能量，也是气的一部分。
你冬天不怕冷——身体产生的热量，还是气在起作用。

**气 = 人体内一切具有推动、温煦、防御、固摄功能的精微物质。**

说白了：**气就是让你"活着"的动力。** 心跳、呼吸、消化、出汗、思考——全靠气在背后推着。

### 气的四大功能

| 功能 | 含义 | 打比方 |
|------|------|--------|
| **推动** | 推动血液运行、推动脏腑工作 | 发动机 |
| **温煦** | 维持体温，让身体不冷 | 暖气片 |
| **防御** | 保护体表不受外邪侵入 | 保安 |
| **固摄** | 管住血液不外溢、汗液不乱出、脏器不下垂 | 阀门 |

### 气的分类

人体的气不是铁板一块，分成几种"部队"，各有分工：

| 气的名称 | 来源 | 驻扎位置 | 职责 | 公司类比 |
|---------|------|---------|------|---------|
| **元气** | 先天之精 + 后天补充 | 肾→全身 | 生命活动的根本动力 | 董事长——总掌舵 |
| **宗气** | 呼吸的清气 + 食物的水谷精气 | 胸中 | 推动呼吸和血液循环 | 运营总监——管"跑腿的" |
| **营气** | 食物精华 | 脉管内，随血运行 | 化生血液，营养全身 | 后勤部——管供给 |
| **卫气** | 食物精华中"剽悍滑利"的部分 | 脉管外，体表 | 保卫体表，控制汗孔开合 | 保安部——巡逻站岗 |

### 气出了问题是什么样？

**气虚——动力不足：**
- 表现：疲乏无力、声低气短、容易出汗、面色淡白
- 类比：手机电量 10%——什么都能干但什么都没劲干

**气滞——动力堵了：**
- 表现：胀痛（特别是两肋、胃脘、腹部），叹气后舒服，情绪郁闷
- 类比：高速公路堵车——总量没少，就是走不动

**气逆——动力方向反了：**
- 表现：咳嗽（肺气上逆）、呕吐（胃气上逆）、打嗝
- 类比：电梯故障往上冲——该降的降不下来

## 二、血——看得见的"营养"

### 血是什么？

血就是中医说的脉管里流动的红色液体，跟现代医学说的血液基本对应。

但中医对血的理解更偏功能：**血是全身组织器官的营养来源。**

### 血的生成

血的原料主要有两个来源：

1. **水谷精微**（食物的营养）→ 经脾的运化 → 上输到心肺 → 变化而赤，成为血
2. **肾精**（先天之精）→ 化生骨髓 → 骨髓生血

所以补血首先要**健脾**（让"工厂"有原料），同时**补肾**（让"底子"够用）。

### 血的运行

血在脉管里循行全身，靠三股力量"护送"：

1. **心气推动**——心是泵
2. **肝气疏泄**——肝保证道路通畅
3. **脾气统摄**——脾管住血不乱跑

三者缺一不可。心气虚了血走不动（心悸、面白）；肝气郁了血也堵了（瘀血、刺痛）；脾气虚了血管不住（出血、紫癜）。

### 气与血的经典关系

这是整个中医理论中最重要的配对关系之一，前面讲过，再强调一次：

> **气为血之帅** —— 气推动血、统摄血、化生血。没有气，血是一潭死水。
> **血为气之母** —— 血承载气、滋养气。没有血，气是无根之火。

**临床意义：**
- 大出血后人为什么虚脱？→ 血脱则气脱——血大量丢失，气也跟着散了。所以抢救大出血，中医不仅补血还要**大补元气**（独参汤救急）。
- 长期气虚的人为什么容易瘀血？→ 气虚无力推血 → 血行缓慢 → 瘀滞。所以活血化瘀方里经常加补气药（黄芪就是常客）。

### 血出了问题是什么样？

**血虚——营养不够：**
- 表现：面色苍白或萎黄、头晕眼花、心悸、手脚麻木、月经量少色淡
- 类比：银行卡余额不足——身体各部门都在"嗷嗷待哺"

**血瘀——营养堵了：**
- 表现：刺痛（固定不移）、面色暗、嘴唇紫暗、舌有瘀斑、肿块
- 类比：水管被水垢堵了——水量没少但流不过去

**血热——血被"烤"了：**
- 表现：出血（鼻血、牙龈血、月经量多）、皮疹、心烦
- 类比：暖气管里的水被烧得太热，管子承受不住开始渗漏

---

## 三、津液——身体里的"水利系统"

### 津液是什么？

津液就是体内一切正常的水分——除了血以外的所有液体。汗、泪、唾液、胃液、关节滑液……都是津液的不同形式。

### 津和液有什么区别？

| | 津 | 液 |
|--|----|----|
| 质地 | 稀薄、清亮 | 浓稠、黏滑 |
| 分布 | 体表、肌肉、孔窍 | 关节、脑髓、骨腔 |
| 功能 | 滋润皮肤、温养肌肉 | 濡养关节、充养脑髓 |
| 打比方 | 喷雾——到处喷洒、随时补充 | 润滑油——渗透到深层、不易消耗 |

### 津液的代谢：多器官接力

津液的"一生"是一场接力赛：

1. **胃** → 接收你喝的水和食物中的水分
2. **脾** → 把水分提取出来，向上输送
3. **肺** → 宣发到体表（变成汗），肃降到下面
4. **肾** → 蒸腾气化——有用的水重新吸收利用，没用的变成尿排掉
5. **膀胱** → 储存尿液，排出体外
6. **三焦** → 水液运行的通道

**任何一个环节出问题，水就会"堵"在某个地方——变成病理产物：痰、饮、水肿。**

比如：
- 脾虚运化不了水 → 水湿停在中焦 → 腹胀、腹泻、身体沉重
- 肺失宣降 → 水停在上焦 → 面浮肿、咳痰多
- 肾阳虚 → 气化不了水 → 下肢水肿、尿少

### 津液与气血的关系

- **津血同源：** 津液和血都来自食物的精微。津液可以渗入脉中补充血液，血液中的水分也可以渗出脉外变成津液。所以大出血时不能猛灌水发汗（"夺血者无汗"），大汗后不能放血（"夺汗者无血"）。
- **气能化津：** 气推动津液的生成、输布、排泄。气虚了，水就运不动——痰饮水肿。
- **津能载气：** 津液也承载着气。大量出汗后人会虚脱——不仅是水分丢了，气也跟着散了（"气随津脱"）。

### 本章小结

气、血、津液是人体运转的三大"物资"。**气是动力（看不见的推手），血是营养（看得见的补给），津液是润滑和冷却（无处不在的水系统）。** 三者互相依存、互相转化，谁出问题都会牵连其他两个。

物资有了，还需要运输网络——接下来看**经络**。

# 第五章：经络学说

> **本章你将学到：** 经络是什么、十二正经的基本规律、为什么针灸能治病。

## 一、经络是什么？

### 高速公路比喻

如果说气血津液是"物资"，那经络就是"运输网络"。

- **经脉 = 高速公路（主干道）：** 纵行于身体上下，连接脏腑和体表
- **络脉 = 支路小巷（分支网络）：** 从主干道分出去，遍布全身每一个角落

经络把五脏六腑、四肢百骸、皮肉筋骨全部联结成一个整体。没有经络，脏腑就是一堆孤立的"部门"，互相联系不上。

### 经络的功能

| 功能 | 含义 | 大白话 |
|------|------|--------|
| 运行气血 | 气血通过经络到达全身 | 物流配送 |
| 联络脏腑 | 把内脏和体表、上下内外连起来 | 内部通信网络 |
| 抗御外邪 | 卫气通过经络分布到体表 | 边防线 |
| 传导感应 | 针灸刺激一个穴位，信号沿经络传到远处 | 电话线 |

---

## 二、十二正经——最重要的12条主干道

十二正经是经络系统的核心，每条经络连接一个脏或腑，走一条固定路线。

### 命名规律

每条经络的名字都包含三个信息：**手/足 + 阴/阳 + 脏/腑名**。

| 分类依据 | 规则 |
|---------|------|
| **手/足** | 经络走手臂的叫"手……经"，走腿的叫"足……经" |
| **阴/阳** | 走在身体内侧（阴面）的叫"阴经"，走外侧（阳面）的叫"阳经" |
| **三阴三阳** | 阴分太阴、少阴、厥阴；阳分阳明、太阳、少阳 |

具体来看：

**手三阴经（从胸走手，走手臂内侧）：**

| 经络 | 连接的脏 | 简记 |
|------|---------|------|
| 手太阴肺经 | 肺 | 拇指侧内面 |
| 手厥阴心包经 | 心包 | 中指方向内面 |
| 手少阴心经 | 心 | 小指侧内面 |

**手三阳经（从手走头，走手臂外侧）：**

| 经络 | 连接的腑 | 简记 |
|------|---------|------|
| 手阳明大肠经 | 大肠 | 食指侧外面 |
| 手少阳三焦经 | 三焦 | 无名指方向外面 |
| 手太阳小肠经 | 小肠 | 小指侧外面 |

**足三阳经（从头走足，走腿外侧）：**

| 经络 | 连接的腑 | 简记 |
|------|---------|------|
| 足阳明胃经 | 胃 | 腿前面 |
| 足少阳胆经 | 胆 | 腿外侧面 |
| 足太阳膀胱经 | 膀胱 | 腿后面 |

**足三阴经（从足走腹/胸，走腿内侧）：**

| 经络 | 连接的脏 | 简记 |
|------|---------|------|
| 足太阴脾经 | 脾 | 腿内侧前面 |
| 足厥阴肝经 | 肝 | 腿内侧中间 |
| 足少阴肾经 | 肾 | 腿内侧后面 |

### 流注顺序——24小时的循环

十二条经络的气血按照固定顺序流注，每条经络"值班"2 小时（一个时辰）：

肺（3-5）→ 大肠（5-7）→ 胃（7-9）→ 脾（9-11）→ 心（11-13）→ 小肠（13-15）→ 膀胱（15-17）→ 肾（17-19）→ 心包（19-21）→ 三焦（21-23）→ 胆（23-1）→ 肝（1-3）→ 又回到肺……

**这就是"子午流注"的基本原理。** 为什么老人凌晨 3-5 点容易咳嗽？因为那是肺经当令，肺有问题在这个时段最容易暴露。

## 三、经络的临床意义

### 为什么针灸能治病？

你可以把经络想成城市的高速公路系统。

- 健康状态：道路畅通，物资（气血）正常运输
- 生病状态：某段路堵了，物资到不了目的地

针灸就是在**堵点或关键节点（穴位）上施加刺激**，疏通道路，恢复气血运行。

**"通则不痛，痛则不通"**——疼痛的本质就是气血不通。把不通的地方打通了，痛自然就消了。

### 循经取穴——为什么头痛要扎手上的穴？

这个问题初学者很困惑：头痛为什么不扎头上，反而扎手上的合谷穴？

因为**经络是一条通路**。手阳明大肠经从手的食指出发，沿着手臂外侧一路向上，经过肩膀、脖子，到达面部和头部。

头痛如果在前额和面部（阳明经的循行区域），沿着这条"高速公路"向下找一个好用的"入口"来疏通——合谷穴就是这条路上最著名的"收费站"，刺激它就能疏通整条经络，头部的堵塞也就解开了。

这就是**循经取穴**——沿着经络走向来选穴位。

### 经络辨证

除了治疗，经络还帮助诊断：

- 疼痛沿着某条经络走向分布 → 提示那条经络出了问题
- 例如：偏头痛在颞侧（太阳穴附近）→ 少阳经的区域 → 可能是少阳证（肝胆问题）
- 例如：后脑勺痛、颈部僵硬 → 太阳经的区域 → 可能是太阳证（外感风寒）

### 本章小结

经络是人体的"通信和运输网络"，把脏腑、气血、体表连成一个整体。十二正经是核心框架，每条连接一个脏腑、走一条固定路线。**理解了经络，你就明白了"为什么扎手上的针能治头痛""为什么脚底按摩能调理脏腑"这些看似神奇的事情——因为它们在同一条线路上。**

人体的"地图"和"运输系统"讲完了。最后一个问题：**人为什么会生病？** 进入第六章。

# 第六章：病因与病机

> **本章你将学到：** 中医认为人为什么会生病（病因），病是怎么一步步发展的（病机）。

## 一、病因——人为什么会生病？

中医把病因分成三大类：外因、内因、不内外因。

### 1. 外因——六淫（六种"坏天气"）

"淫"在这里是"过度"的意思。正常的风寒暑湿燥火是自然界的六种气候，叫"六气"——它们本身不会让人生病。但当它们**太过、不及、或出现在不该出现的季节**时，就变成了致病因素，叫"六淫"。

| 六淫 | 特性 | 让你想到什么 | 致病特点 |
|------|------|-------------|---------|
| **风** | 善行数变、轻扬向上 | 一阵风——来得快、到处跑、变来变去 | 发病急、症状游走不定、多犯上部（头面） |
| **寒** | 收引凝滞 | 冬天——缩手缩脚、什么都冻住 | 怕冷、疼痛（冷了就缩、缩了就疼）、气血凝滞 |
| **暑** | 炎热升散、多挟湿 | 大夏天——又热又闷 | 高热、大汗、口渴，常伴胸闷恶心 |
| **湿** | 重浊黏滞、趋下 | 梅雨季节——黏黏糊糊、甩不掉 | 身体沉重、头昏如裹、病程缠绵不愈 |
| **燥** | 干涩、伤津 | 北方秋天——皮肤干裂、嗓子冒烟 | 口干、皮肤干燥、干咳少痰 |
| **火（热）** | 炎上、伤津、动血 | 一把火——往上烧、烧干水、烧破血管 | 高热、口渴、出血、神志不清 |

**记忆技巧：** 想想你在不同天气里的感受——吹风头痛（风）、冬天手疼（寒）、桑拿天胸闷（暑湿）、秋天嘴唇裂（燥）、发烧口渴（火）——这就是六淫的致病特点。

**"风为百病之长"：** 风邪最特殊，它经常是"带头大哥"——不单独来，拉着别的邪气一起入侵：风寒、风热、风湿……所以感冒最常见。

### 2. 内因——七情（情绪致病）

中医很早就认识到：**情绪不仅影响心情，还会伤害身体。**

| 七情 | 对应的脏 | 致病机理 | 生活中的例子 |
|------|---------|---------|-------------|
| **喜** | 心 | 喜则气缓——过度兴奋让气散了 | 范进中举，喜极发狂 |
| **怒** | 肝 | 怒则气上——暴怒让气往上冲 | 气得头疼、脸红脖子粗、血压飙升 |
| **忧/悲** | 肺 | 悲则气消——过度悲伤让气消耗殆尽 | 林黛玉忧愁过度，肺病缠身 |
| **思** | 脾 | 思则气结——想太多让气结在中焦 | 考试前焦虑，吃不下饭 |
| **恐** | 肾 | 恐则气下——恐惧让气往下坠 | 吓得尿裤子（肾管膀胱，气下坠则失禁） |
| **惊** | 心 | 惊则气乱——突然受惊让气乱了 | 被吓到后心跳加速、手足无措 |

**注意：** 正常的情绪是健康的，问题出在**过度或持久**。偶尔生气不会病，天天生气就要出事。偶尔焦虑是正常的，整天焦虑就会伤脾。

**核心规律：** 每种情绪都会让气发生特定方向的异常——上、下、散、结、消、乱。气一乱，对应的脏就受害。

### 3. 不内外因——其他致病因素

不属于外界气候，也不属于情绪，但也会让人生病：

| 因素 | 举例 |
|------|------|
| **饮食不节** | 暴饮暴食伤脾胃、过食辛辣生内热、过食生冷伤阳气 |
| **劳逸失度** | 过劳伤气（"劳则气耗"）、过逸伤脾（"久卧伤气"）、房劳伤肾 |
| **外伤** | 跌打损伤、虫兽咬伤 |
| **痰饮、瘀血** | 这两个很特殊——它们既是病的**结果**，又能成为新的**原因**（恶性循环） |

**痰和瘀血特别值得说一下：**

痰是水液代谢失常产生的病理产物。但痰一旦形成，它会到处跑、到处堵——堵在肺就咳痰，堵在头就头晕，堵在心就神志糊涂，堵在经络就麻木。所以中医有句话：**"百病多由痰作祟。"**

瘀血是血液运行不畅形成的停滞。它不仅是症状，还会变成新的致病因素——堵在哪里，哪里就痛、就长东西。

## 二、病机——病是怎么发展的？

知道了"为什么生病"（病因），还要知道"生了病以后身体里发生了什么"（病机）。

### 1. 邪正斗争——入侵者 vs 免疫系统

中医把致病因素叫**"邪气"**，把人体的抵抗力叫**"正气"**。

> **"正气存内，邪不可干；邪之所凑，其气必虚。"**

翻译：你自身抵抗力够强，病邪就进不来。反之，你之所以生病，一定是在某个方面虚了。

**生病的本质 = 邪气和正气的博弈。**

| 局面 | 表现 | 临床所见 |
|------|------|---------|
| 正强邪弱 | 正气占上风，邪气被赶走 | 感冒很快好了，不用吃药 |
| 邪正相争 | 势均力敌，打得热火朝天 | 高热、炎症——战斗正激烈 |
| 邪盛正虚 | 邪气占上风 | 病情加重、迁延不愈 |
| 正虚邪恋 | 正气太弱连仗都打不起来 | 低热、乏力——慢性病、缠绵不愈 |

**治疗策略：**
- 邪气盛 → 祛邪为主（把入侵者赶走）
- 正气虚 → 扶正为主（先增强抵抗力）
- 实际操作中往往 **扶正与祛邪并用**，根据主次调整比例

### 2. 阴阳失调——回扣第一章

所有的病机归根结底都可以用阴阳来概括：

| 失调类型 | 机制 | 表现 |
|---------|------|------|
| **阴偏盛** | 阴邪入侵，阴太多 | 实寒证：怕冷、腹泻、面白 |
| **阳偏盛** | 阳邪入侵，阳太多 | 实热证：高热、口渴、面红 |
| **阴偏衰（阴虚）** | 阴液不足，管不住阳 | 虚热证：五心烦热、盗汗、口干 |
| **阳偏衰（阳虚）** | 阳气不足，管不住阴 | 虚寒证：畏寒、四肢冷、乏力 |
| **亡阴** | 阴液急剧耗竭 | 危重：大汗黏手、烦躁、脉细数 |
| **亡阳** | 阳气急剧外脱 | 危重：大汗淋漓（冷汗）、四肢冰冷、脉微欲绝 |

### 3. 气血失常——回扣第四章

| 失常类型 | 机制 | 常见表现 |
|---------|------|---------|
| 气虚 | 气不够了 | 乏力、气短、自汗 |
| 气滞 | 气堵了 | 胀痛、情绪郁闷 |
| 气逆 | 气走反了 | 咳嗽、呕吐、呃逆 |
| 气陷 | 气该升不升 | 脏器下垂、久泻不止 |
| 血虚 | 血不够了 | 面白、头晕、心悸 |
| 血瘀 | 血堵了 | 刺痛、肿块、面色暗 |
| 血热 | 血被热逼了 | 出血、皮疹、烦躁 |
| 气滞血瘀 | 气堵→血也堵 | 胀痛+刺痛并见 |
| 气虚血瘀 | 气弱→推不动血 | 乏力+局部疼痛 |
| 气不摄血 | 气弱→管不住血 | 慢性出血（便血、崩漏） |

### 本章小结

病因回答"是什么让你生病了"——外来的六淫、内生的七情、以及饮食劳逸等。病机回答"生病后身体里发生了什么"——邪正交争、阴阳失调、气血失常。

**理解了病因病机，你就掌握了中医的"诊断逻辑"：看到症状 → 推导出病因和病机 → 确定治法。** 这就是中医"辨证论治"的核心流程。

```
│    病因（为什么病）  ──→  病机（怎么发展）                │
│         ↓                                               │
│    辨证论治（看到什么 → 判断什么 → 怎么治）               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**自下而上读：**

1. **阴阳**告诉你：世界是动态平衡的，有阴有阳，失衡就生病。
2. **五行**告诉你：平衡不是两个人的事，是五方互动——相生相克构成网络。
3. **藏象**告诉你：这个网络在人体里的具体落地——五脏六腑各司其职。
4. **气血津液**告诉你：脏腑靠什么运转——气的推动、血的滋养、津液的润滑。
5. **经络**告诉你：这些物资走哪条路——十二经脉是主干道，络脉是毛细路网。
6. **病因病机**告诉你：哪些因素会打破平衡，打破之后身体内部会发生什么。

**而中医的终极方法论——辨证论治——就是把以上所有知识综合运用：**

> 看到一组症状 → 用阴阳判断寒热虚实 → 用五行和藏象定位到具体脏腑 → 用气血津液分析物质基础的变化 → 用经络确定传导和治疗路径 → 用病因病机推导出疾病的来龙去脉 → 得出治法和方药。

**这就是完整的链条。学明白了这六大学说，你就拿到了打开中医大门的全套钥匙。**
